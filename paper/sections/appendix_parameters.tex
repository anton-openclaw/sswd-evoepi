\section{Parameter Tables}
\label{sec:appendix_parameters}

Table~\ref{tab:params_disease}--\ref{tab:params_pathogen} list all 47
parameters included in the sensitivity analysis, grouped by module.
For each parameter we report the default value used in validation runs,
the sensitivity analysis range (low--high), the distribution used for
sampling (U = uniform, LU = log-uniform, D = discrete), and the
empirical confidence rating:
$\star\star\star$ = directly measured,
$\star\star$ = constrained by related data,
$\star$ = poorly constrained or assumed.
