# Forward Projection Experiments (2012–2050)

All scenarios use W330 best-calibrated parameters as baseline.
K=1000 with K_ref=5000 density scaling. 38 years total (2012–2050), disease at year 1 (2013).

## Scenarios

| ID | Name | Description |
|----|------|-------------|
| F01 | **Baseline** | W330 unchanged, run to 2050. What happens with no intervention? |
| F02 | **Disease removal 2018** | Pathogen eliminated at year 6 (2018). Counterfactual: what if we cured SSWD? |
| F03 | **No pathogen evolution** | Pathogen can't cold-adapt (pathogen_adaptation=false). How important is coevolution? |
| F04 | **Accelerated warming** | SST trend doubled (0.04°C/yr vs 0.02). Climate change amplifies disease? |
| F05 | **No gradient mechanisms** | δ_env(T) and P_env gating OFF. What would the old model predict to 2050? |
| F06 | **Disease removal 2025** | Pathogen eliminated at year 13 (2025/now). Recovery from current state? |

## Key Questions
1. Does the model reach equilibrium or keep declining?
2. How fast do populations recover without disease?
3. Does pathogen evolution eventually overwhelm host resistance?
4. Does warming shift the gradient northward over time?
5. How different is recovery if we intervene now (2025) vs 2018?
